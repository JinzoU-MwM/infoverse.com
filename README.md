# InfoVerse

Fresh Next.js App Router project for the InfoVerse media portal.

## Stack
- Next.js + TypeScript + Tailwind CSS
- Drizzle ORM + SQLite
- Better Auth (email/password, role-based admin guards)

## Commands
- `pnpm install`
- `pnpm db:seed`
- `pnpm dev`
- `pnpm test`
- `pnpm lint`
- `pnpm build`

## Default Admin Credentials
Set in `.env.local` (see `.env.example`).
